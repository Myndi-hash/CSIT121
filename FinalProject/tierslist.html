In this module I learned about what is a table, and what is a Form. I also learned how they can be styled using CSS and how to create both, a table and a Form. An HTML Table is a well structured set of data that is made up of rows and columns, represented in a tabular format. HTML table should be used for tabular data type not for web pages lay out. An HTML Form is used to collect user input. The data collected from the user can be send to a server for processing.
